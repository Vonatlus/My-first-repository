• git config --global user.name blabla - твоє імя, яке відображається в commit
  • git config --global user.email blabal@bla - твій email, який відображається в commit
  • git init - створюєм сховище
  • git status- перелік всіх змін, що потребують add
  • git add- додаєм зміни для commit
  • git commit - створює історію
  • git log- відображає всю історію
  • git checkout - переходим на обранну гілку
  • git branch - стрворюєм гілку
  • git push - закидуєм в віддаленне сховище
  • git pull - качаєм з віддаленого сховища
  • git merge - об'єднує декілька гілок в одну
  • git clone - дублюєм все з сховища
  • git revert - створює commit, який скасовує зміни зроблені іншим commit
  • git reset - скасовує add, проте лишає файл незмінним
  • git rebase - повторює, переміщує зніни однієї гілки на іншу
  • git cherry-pick - створює коміт і їх еквіваленти на інших гілках, який вносить зміни
  • git stash - тимчасово зберігає всі незаcommit зміни

.gitignore - приховує всі файли які автоматизовано генеруються
